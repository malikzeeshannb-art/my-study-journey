# Responsive Restaurant Landing Page – FreshBite

This project is a small responsive restaurant landing page built with **HTML and CSS** as part of my frontend learning journey.

Instead of following step-by-step tutorials, I practice by asking AI to act as a **senior developer or client** and give me a project brief. I then build the project independently based on the requirements.

This project focuses on practicing **layout techniques, responsiveness, and UI polish**.

---

## Project Overview

The page represents a simple restaurant landing layout called **FreshBite**.
It contains a navigation header, hero section, featured menu cards, an about section, and contact information.

The goal was to create a **clean, responsive card layout** that works across different screen sizes while improving my understanding of layout behavior.

---

## Key Features

• Sticky navigation header
• Responsive card layout using **CSS Grid**
• Flexbox used for navigation layout
• Hover interactions on menu cards
• Smooth layout scaling across screen sizes
• Simple anchor navigation (Menu / About / Contact)

---

## Skills Practiced

HTML Structure
CSS Layouts (Grid + Flexbox)
Responsive Design Principles
UI Hover Effects
Semantic Sectioning

Although CSS Grid is commonly used for card layouts, I intentionally experimented with layout behavior and responsive sizing to strengthen my understanding of CSS layout systems.
